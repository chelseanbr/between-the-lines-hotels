I am a data scientist with an electrical/product engineering background and 2+ years experience in programming and data analysis. I created this web app to demonstrate how well Airbnb review sentiments can be classified with a model I trained using scraped Tripadvisor data. I did this because I noticed sites like Amazon, Yelp, and Tripadvisor make it easy to separate negative and positive reviews, but Airbnb does not. With futher improvements, I hope to make it easier to identify, filter, and quantify positive, neutral, or negative reviews on Airbnb so you may not have to read every single review blindly and can pick a place faster! :)

I trained a CNN-LSTM neural network using Tripadvisor hotel reviews with ratings.
For more details, you can check out the GitHub repo for this project.

Thanks for checking out my Airbnb sentiment classifier!
